# Titanic Survival Prediction (Machine Learning Project)

📌 Project Overview:

This project focuses on predicting passenger survival on the Titanic using machine learning techniques. The objective is to build and compare multiple models, identify the best-performing approach, and ensure strong generalization on unseen data.

🔍 Key Contributions:

1. Developed an end-to-end machine learning workflow, including data preprocessing, feature engineering, model training, and evaluation.
2. Performed feature engineering by handling missing values, encoding categorical variables, and creating new features (e.g., titles, family size).
3. Trained and compared multiple models, including Logistic Regression, Decision Trees, and Random Forest, to identify the optimal model.
4. Implemented proper validation techniques to avoid data leakage and ensure robust performance on unseen test datasets.

📊 Business Impact & Insights:

1. Model Performance: Tree-based models (e.g., Random Forest) demonstrated superior performance in capturing non-linear relationships in the data.
2. Feature Importance: Engineered features such as passenger titles and family size significantly improved prediction accuracy.
3. Generalization: Ensured reliable model performance by strictly separating training and test datasets, reflecting real-world prediction scenarios.

🏆 Results:

- Achieved **1st place among 10+ teams in a class-wide competition**
- Final evaluation was based on prediction accuracy on unseen test datasets provided by the instructor
